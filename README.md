# Sobre o projeto

Desafio prático sobre a implementação de testes unitários utilizando JUnit e TDD (Test-Driven Development).

# Tecnologias utilizadas
## Metodolodias
- TDD

## Back end
- Java
- JUnit 5
- Maven
